The screen helper captures the visible screen as a PNG, keeps only a small recent buffer in memory, and sends screenshots to OpenAI vision only when the user asks for help.

The model is prompted to explain the screen in plain, respectful language for an older adult. It should mention only visible details, avoid repeating sensitive information, and give one safe next step when useful.

The model name is configurable with VISION_MODEL. The default is gpt-5.4-nano for cost control; use gpt-5.4-mini if screen understanding needs to be stronger.
